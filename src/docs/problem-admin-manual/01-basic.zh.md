# 新增題目頁面導覽

本手冊提供新增題目頁面的整體結構說明，幫助您快速了解各區塊的功能定位。

---

## 主頁介紹

1. 題目名稱：必填欄位，用於識別題目。

2. 顯示狀態：
   - **顯示**：學生可在課程中看到此題目
   - **隱藏**：題目學生不可見
3. 模式切換：
   - **一般模式**：僅顯示基礎設定區塊
   - **進階模式**：額外顯示「判題流程」與「資源檔案」區塊
     > 系統會記住您的偏好設定，下次開啟頁面時會自動套用。

---

## 五大區塊介紹

### 1. 題目敘述

編輯題目的詳細說明，包含：

- 題目說明
- 輸入、輸出格式
- 題目提示
- 範例輸入/輸出

### 2. 配置設定

設定題目的提交規範與功能開關：

- 支援語言（C / C++ / Python）
- 標籤管理
- 提交額度管理
- 繳交格式
- 測試模式
- AI 助教功能
- 網路與環境擴充
- 產物收集

### 3. 判題流程（進階）

設定程式執行環境的進階選項：

- 檔案存取權限
- 函式庫使用限制
- 執行模式選擇
- 自訂批改腳本 / 計分腳本

### 4. 測試資料

上傳測資壓縮檔並調整各任務參數：

- 時間限制
- 記憶體限制
- 各任務分數

### 5. 資源檔案（進階）

上傳學生或教師專用的資源檔案（如 bmp、csv）。

---

## 錯誤摘要與提交

### 錯誤摘要

當表單驗證失敗時，頁面底部會顯示所有錯誤的摘要清單。點擊任一錯誤可自動跳轉至對應區塊。

### 提交狀態

提交按鈕下方會根據表單狀態顯示不同訊息：

- **等待編輯**：尚未嘗試提交
- **提交受阻**：存在驗證錯誤，需修正
- **準備就緒**：所有欄位驗證通過

---

## 快速操作流程

1. 填寫題目名稱
2. 展開「題目敘述」撰寫題目內容
3. 展開「配置設定」設定語言與提交規範
4. 展開「測試資料」上傳測資壓縮檔
5. 依需求開啟進階模式設定判題流程
6. 點擊提交按鈕完成建立

> **提示**：建議先上傳測資，系統會自動解析任務結構供後續設定使用。
